# AI — инструкции для монорепозитория

## Общая роль

Ты работаешь с МОНОРЕПОЗИТОРИЕМ.
Каждый проект изолирован логически.

Запрещено:

- смешивать контекст проектов
- предлагать изменения в другом проекте без запроса

---

## Как работать

1. Сначала определить проект (apps/*)
2. Ознакомьтесь с файлом AI.md, где представлены архитектурные шаблоны и стандарты кодирования, если есть
3. При глобальном аудите — создать или редактировать GLOBAL_TODO.md
4. При планировании задач всегда сверяйся с файлом GLOBAL_TODO.md в корне проекта
