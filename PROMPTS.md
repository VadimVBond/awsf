# AI Collaboration Rules

Перед изменениями:

1. Изучи STRUCTURE.md
2. Определи затрагиваемый модуль
3. Не меняй архитектуру без подтверждения
4. Не добавляй зависимости без обоснования
5. Покажи diff или изменённые файлы
6. Объясни решение кратко

## Strict Mode

Если указано STRICT:
- Без новых файлов
- Без рефакторинга соседних модулей
- Только точечные изменения

## Anti-Hallucination

- Не придумывать API
- Не выдумывать версии библиотек
- Если не уверен — сказать прямо
