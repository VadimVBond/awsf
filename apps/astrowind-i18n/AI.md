# AI.md

Данный файл является ЕДИНСТВЕННЫМ источником правил для проекта.
Любые альтернативные интерпретации запрещены.

## Область действия

- Правила применяются ко всему репозиторию.
- Исключений нет.

## Стек проекта

- Astro
- Шаблон AstroWind
- TypeScript
- i18n через словари переводов (*.ts)
- Tailwind CSS и Bootstrap (существующее смешанное использование)

## Страницы

- Все страницы расположены в каталоге src/pages.
- Страницы собираются ИСКЛЮЧИТЕЛЬНО из существующих блоков AstroWind.
- Создание кастомных layout’ов или страниц «с нуля» запрещено.
- Дублирование разметки AstroWind блоков запрещено.

## Компоненты

- Используются только существующие компоненты проекта.
- Изменение внутренней логики компонентов запрещено без явного указания.
- Создание новых абстракций без запроса запрещено.

## i18n и контент

- Текстовый контент не должен быть захардкожен.
- Все переводы хранятся только в словарях переводов (*.ts).
- Inline-переводы в компонентах и страницах запрещены.
- Язык страницы определяется существующей i18n-логикой проекта.

## Стили и ассеты

- Используется ТОЛЬКО существующий подход к стилям.
- Добавление новых CSS-фреймворков запрещено.
- Изменение стратегии Tailwind / Bootstrap запрещено.
- Inline-стили запрещены, если они отсутствуют в текущем коде.

## Запрещено

- Вводить новые архитектурные паттерны.
- Изменять структуру проекта.
- Оптимизировать, рефакторить или «улучшать» код без прямого запроса.
- Добавлять «best practices», не подтверждённые текущим кодом проекта.

## Терминология

- Слово **"Checklist"** (Чек-лист/список задач) и связанные с ним списки всегда переводить на русский язык, как в коде (в интерфейсах/контенте), так и в ответах ассистента.
