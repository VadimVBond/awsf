# Templates GEMINI Protocol

Правила для шаблонов сайтов (Astro / HTML demo).

---

# 1. Общие принципы
- Шаблон — пример, не менять глобальные зависимости.
- Не дублировать код в нескольких шаблонах.
- Использовать shared-ui компоненты из libs при необходимости.

---

# 2. Thinking Phase
- Проверить package.json и сборку.
- Проверить роутинг и файлы стилей.
- Проверить использование компонентов.

---

# 3. Архитектура
- Каждый шаблон — самостоятельный проект.
- Возможность переноса в отдельный репозиторий без изменений.

---

# 4. Безопасность
- Не хранить секреты и приватные ключи.
